# Análisis de Apuestas de Fútbol con IA

Dashboard web para buscar partidos de fútbol por liga, consultar datos deportivos y generar análisis de apuestas con inteligencia artificial.

> Proyecto en desarrollo. Actualmente la interfaz usa datos mock/simulados. Más adelante se integrará API-Football y OpenAI.

---

## Objetivo del proyecto

Crear una plataforma sencilla y efectiva para analizar partidos de fútbol usando datos deportivos y apoyo de IA.

La aplicación permitirá:

- Seleccionar ligas específicas.
- Buscar partidos por temporada, fecha y estado.
- Consultar datos relevantes del partido.
- Revisar información como forma reciente, lesiones, sanciones, alineaciones, cuotas, standings y head to head.
- Generar análisis con IA.
- Mostrar mercados sugeridos, riesgos principales, predicción prudente y advertencia de apuesta responsable.

---

## Ligas consideradas

Por ahora el proyecto está enfocado únicamente en estas ligas:

- La Liga (España)
- Superliga China
- Bundesliga (Alemania)
- Primeira Liga (Portugal)
- Ligue 1 (Francia)
