# Jugador con posible gol - Fase 2

La ausencia de un jugador en un partido de la muestra no se interpreta como lesion o sancion. Solo se marca baja si la fuente de lesiones/sanciones lo reporta.

Cada candidato expone:

- partidos evaluados;
- partidos jugados;
- minutos;
- goles;
- tiros;
- tiros a puerta;
- xG si API-Football lo entrega;
- probabilidad estimada conservadora;
- calidad de muestra;
- advertencias.

Si API-Football no entrega xG individual, se muestra como dato no disponible y no se inventa.

La probabilidad estimada de gol es una referencia conservadora basada en tiros, tiros a puerta, minutos, goles/penales recientes y expectativa ofensiva del equipo. No reemplaza cuotas ni debe tratarse como probabilidad oficial.
